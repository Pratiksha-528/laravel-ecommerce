# Laravel E-Commerce Backend

## Features
- Admin Login (Session-based)
- Product CRUD with multiple image support
- Cart Management
- API Exception Handling
- Postman Collection Included

## Requirements
- PHP 8.0+
- MySQL
- Composer
- Node.js + npm
